# Proyecto de tienda en línea: MexShop

### Stack
### Front: React/Vite + TailwindCSS + Axios
### Back: Node.js + Express + TypeScript + Tokens JWT + CORS
### BDD: Prisma + MySQL

## Instrucciones:
Esta es una demo de una tienda en línea, donde se hace práctica de habilidades Full-Stack. 
Se puede registrar e iniciar sesión, agregar productos del carrito, usar filtros y una simulación de compra.
